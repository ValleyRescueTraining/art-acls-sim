VRT

Valley Rescue ACLS Simulator

Two-screen browser version for student monitor and instructor control.

Session

Use the same session code on both devices.

What this starter does

  • Hosts clean student and instructor pages on GitHub Pages
  • Uses Firebase Realtime Database to sync patient state live
  • Keeps ACLS controls simple: rhythm, vitals, perfusion, CPR, pacing, shock
  • Gives you a clean base to improve waveforms and sounds

Setup checklist

  1. Create a GitHub repo and upload these files.
  2. Turn on GitHub Pages for the repo.
  3. Create a Firebase project and Realtime Database.
  4. Paste your Firebase config into firebase-config.js.
  5. Open student.html?session=demo-alpha on one device and instructor.html?session=demo-alpha on another.
